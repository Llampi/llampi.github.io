Proyecto Interacción Humano Computador CC451

Descripción:

Esta aplicación permite a un usuario crear una lista de tareas pendientes a realizar durante el día de manera constante para crear hábitos a fin de superarse en ámbitoss como los estudios, alimentación saludable o ejercicios físicos. También se puede usar para eliminar hábitos negativos del usuario y, de este modo, el usuario puede crear un plan de superación personal a largo plazo.
